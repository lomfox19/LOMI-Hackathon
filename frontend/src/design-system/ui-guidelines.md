PROJECT DESIGN SYSTEM

This project follows a consistent AI SaaS design theme.

COLOR PALETTE

Background: #F5F1EA
Primary Accent: #0F3D2E
Secondary Accent: #2E7D5B
Hover Accent: #0A2B21

Typography

Heading Font: Playfair Display
Body Font: Inter

UI STYLE

* Minimal premium AI SaaS design
* Rounded cards (16px radius)
* Subtle shadows
* Clean spacing
* Glass-style cards

AI VISUAL STYLE

The interface should clearly feel like an **AI-powered platform**.

Add subtle futuristic AI visual cues such as:

* Neural network patterns
* Abstract circuit lines
* AI brain / robotic style icons
* Subtle glowing gradients
* AI wave or signal visuals
* Data flow inspired UI elements

These elements should be **subtle and elegant**, not overwhelming.
The goal is that when users open the website they immediately feel it is an **AI technology platform**.

Avoid overly complex illustrations — keep the design **clean, modern, and AI-focused**.

ANIMATIONS

* Smooth fade-in on scroll
* Card hover lift animation
* Gradient button hover
* Framer Motion animations

AI ANIMATION STYLE

Animations should feel **intelligent, smooth, and futuristic**, such as:

* Subtle glowing hover effects
* Neural pulse style animations
* Soft gradient movement
* Floating AI indicator effects
* Smooth micro-interactions on buttons and cards

COMPONENT RULES

All new UI components must:

1. Follow the same color palette.
2. Use Tailwind classes from the theme.
3. Use Framer Motion for animations.
4. Maintain same spacing and card design.

Never introduce a new color palette or different theme.

All new pages must follow the same layout structure used in the landing page and dashboard.
