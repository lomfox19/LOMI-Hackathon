PROJECT DESIGN SYSTEM

This project follows a consistent **modern AI SaaS design theme** with futuristic visual cues and intelligent UI interactions.

COLOR PALETTE

Background: #F5F1EA  
Primary: #003049  
Secondary: #2E7D5B  
Accent Gold: #FCBF49  
Accent Orange: #F77F00  
Alert Red: #D62828

Color Usage Guidelines

Primary (#003049)
Used for main UI actions, navigation, buttons, and key highlights.

Secondary (#2E7D5B)
Used for secondary actions, UI accents, and supporting elements.

Accent Gold (#FCBF49)
Used for premium highlights, AI indicators, and special UI emphasis.

Accent Orange (#F77F00)
Used for active states, hover highlights, and call-to-action elements.

Alert Red (#D62828)
Used only for alerts, warnings, or destructive actions.

Never introduce colors outside this palette.

Typography

Heading Font: Playfair Display  
Body Font: Inter

Headings should feel elegant, bold, and premium.  
Body text should remain clean, modern, and highly readable.

UI STYLE

* Minimal premium AI SaaS interface
* Rounded cards (16px radius)
* Smooth shadows
* Clean spacing
* Glass-style cards
* Modern layered depth
* Balanced white space
* Premium dashboard feel

Cards should appear slightly elevated with soft shadows to create a floating UI effect.

AI VISUAL STYLE

The interface should clearly feel like an **AI-powered platform**.

Add subtle futuristic AI visual cues such as:

* Neural network patterns
* Abstract circuit lines
* AI brain / robotic style icons
* Subtle glowing gradients
* AI wave or signal visuals
* Data flow inspired UI elements
* Floating system nodes
* Intelligent indicator lights

These elements should be **subtle and elegant**, not overwhelming.

The goal is that when users open the website they immediately feel it is an **AI technology platform**.

Avoid overly complex illustrations — keep the design **clean, modern, and AI-focused**.

ADVANCED VISUAL DEPTH

Use layered depth to make the interface feel modern.

Examples:

* Soft ambient shadows
* Elevated card depth
* Smooth hover elevation
* Subtle gradient lighting
* Inner glow accents

Cards should feel light and modern, not heavy.

GLASSMORPHISM STYLE

Some UI elements may use a glass-style interface.

Characteristics:

* Subtle background blur
* Soft transparency
* Light borders
* Smooth lighting gradients

Glass elements should feel premium and minimal.

ANIMATIONS

* Smooth fade-in on scroll
* Card hover lift animation
* Gradient button hover
* Framer Motion animations
* Smooth page transitions
* Staggered component loading

Animations should feel **smooth and intelligent**, not distracting.

ADVANCED MICRO-INTERACTIONS

Add subtle micro-interactions to improve user experience.

Examples:

* Hover glow on buttons
* Icon reaction animations
* Smooth input focus highlight
* Soft ripple click effects
* Button hover gradient shift

Animation duration should typically remain **200–400ms**.

AI ANIMATION STYLE

Animations should feel **intelligent, smooth, and futuristic**, such as:

* Neural pulse style animations
* Soft gradient movement
* Floating AI indicator effects
* Data flow animation lines
* Signal pulse visuals
* Intelligent loading animations
* Subtle glow pulses
* Dynamic highlight motion

The UI should feel like a **living AI system interface**.

SCROLL ANIMATIONS

Scrolling interactions should feel smooth and modern.

Examples:

* Fade-in sections
* Slide-up card appearance
* Staggered grid animations
* Smooth parallax background movement
* Dynamic gradient lighting shifts

Animations must remain subtle and elegant.

AI LIGHTING EFFECTS

Add subtle lighting effects to enhance the AI feel.

Examples:

* Soft glow around important UI elements
* Gradient highlight edges
* Ambient UI lighting
* Hover illumination effects
* Subtle reflective surfaces

Lighting should feel modern and premium.

COMPONENT RULES

All new UI components must:

1. Follow the same color palette.
2. Use Tailwind classes from the theme.
3. Use Framer Motion for animations.
4. Maintain same spacing and card design.
5. Follow the same typography rules.

Never introduce a new color palette or different theme.

All new pages must follow the same layout structure used in the landing page and dashboard.

DESIGN GOAL

The final experience should feel like:

* A premium AI platform
* A modern SaaS dashboard
* A futuristic intelligent interface

Users should immediately feel they are interacting with a **powerful AI system**.
